# Fellbacher Wines E-Commerce Platform

Этот проект представляет собой современную E-Commerce платформу для винодельни Fellbacher, построенную на cutting-edge технологиях веб-разработки. Приложение сфокусировано на премиальном пользовательском опыте, высокой производительности и эстетичном дизайне.

## 🛠 Технологический Стек

Мы используем самые современные инструменты для обеспечения скорости, надежности и масштабируемости:

- **Next.js 15 (App Router)**: Фреймворк React для server-side rendering, оптимизации маршрутизации и SEO. Используется новая архитектура App Router для лучшей организации данных и компонентов.
- **React 19**: Библиотека для построения пользовательских интерфейсов.
- **TypeScript**: Строгая типизация для повышения надежности кода и удобства разработки.
- **Tailwind CSS 4**: Утилитарный CSS-фреймворк для быстрой и гибкой стилизации. Используется для создания уникальной дизайн-системы.
- **Framer Motion**: Библиотека для создания плавных и сложных анимаций, оживляющих интерфейс.
- **Lucide React**: Набор красивых и легковесных иконок.
- **Context API**: Управление глобальным состоянием (Корзина, Список желаний, Аутентификация) без перегрузки сторонними библиотеками.
- **i18n (Custom)**: Легковесная система интернационализации для поддержки немецкого (DE) и английского (EN) языков.

## 🚀 Подробное описание ключевых модулей

### 🧩 Контексты и Состояние
- **`AuthContext.tsx`**: Обработка авторизации, хранение данных пользователя и имитация API входа.
- **`CartContext.tsx`**: Управление корзиной покупок: добавление, удаление, изменение количества и расчет итоговой стоимости.
- **`WishlistContext.tsx`**: Реализация списка избранного, позволяющая сохранять товары для будущих покупок.
- **`OrdersContext.tsx`**: Модуль управления историей заказов пользователя с отслеживанием статусов доставки.

### 🎨 Компоненты и Интерфейс
- **`Header`**: Универсальная шапка с многоуровневым меню, умным поиском и превью корзины.
- **`Footer`**: Детализированный подвал с картой сайта, контактными данными и часами работы.
- **`WineCard`**: Интерактивная карточка товара с плавными переходами и детальной информацией.

### 🌍 Глобальная Логика
- **`i18n.tsx`**: Продвинутая локализация, позволяющая мгновенно переключать интерфейс между немецким и английским языками.
- **`data/wines.ts`**: Единственный источник правды («Single Source of Truth») для всех данных о винах.
